# Cave Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A Xaero-style cave view — when enclosed, both maps cut away the roof above the player, draw the cave floors below, and blend a depth gradient over the real block colours. Plus a keybind language file so all binds are named/rebindable.

**Architecture:** Reuse the v1.6.0 surface-at-cut renderer. A new `MapMode.CAVE` adds a height-based depth blend in the pure `MapTileRenderer`. A pure `CaveDetector` handles activation (AUTO/ON/OFF + debounce). The client tick scans the real world above the player for a roof; when cave-active the maps place the cut just under that roof and render in CAVE mode. Precedence: manual slice > cave cut > eye-level follow.

**Tech Stack:** Fabric 1.21.11, Java 21, JUnit 5, Gson.

**Build/test:** `$env:JAVA_HOME="<project-root>\libs\jdk21\jdk-21.0.11+10"; .\gradlew build`
Develop directly on `main` (project convention; no worktrees).

```java
package com.mia.aperture.map;

public final class CaveDetector {
    public static final int DEBOUNCE_TICKS = 8;

    private boolean stable = false;
    private int count = 0;

    // Flip the stable state only after DEBOUNCE_TICKS consecutive ticks of the new
    // value; any tick matching the current stable value resets the counter.
    public boolean debounce(boolean raw) {
        if (raw == stable) {
            count = 0;
            return stable;
        }
        count++;
        if (count >= DEBOUNCE_TICKS) {
            stable = raw;
            count = 0;
        }
        return stable;
    }

    public boolean isStable() {
        return stable;
    }

    public static boolean caveActive(MapSettings.CaveMode mode, boolean enclosed) {
        return switch (mode) {
            case ON -> true;
            case OFF -> false;
            case AUTO -> enclosed;
        };
    }
}
```

```java
        return com.mia.aperture.map.MapGeometry.shiftY(roofWorldY - 1, sector);
    }

    // Precedence: manual slice > cave roof-cut > eye-level follow.
    public static int effectiveBandTop(int playerWorldY, int sector, boolean caveActive,
            boolean depthActive, double cutAbyssY, boolean roofFound, int roofWorldY) {
        if (depthActive) {
            return mapBandTopShifted(playerWorldY, sector, true, cutAbyssY);
        }
        if (caveActive && roofFound) {
            return caveCutShiftedY(roofWorldY, sector);
        }
        return mapBandTopShifted(playerWorldY, sector, false, cutAbyssY);
    }
```

```java
    private static void scanEnclosure(Minecraft client) {
        var p = client.player;
        int px = (int) Math.floor(p.getX());
        int pz = (int) Math.floor(p.getZ());
        int headY = (int) Math.floor(p.getEyeY());
        boolean found = false;
        int roofWorldY = AbyssMapState.caveRoofWorldY;
        var pos = new net.minecraft.core.BlockPos.MutableBlockPos();
        for (int dy = 1; dy <= 48; dy++) {
            pos.set(px, headY + dy, pz);
            var bs = client.level.getBlockState(pos);
            if (bs.blocksMotion()) {
                found = true;
                roofWorldY = headY + dy;
                break;
            }
        }
        AbyssMapState.caveEnclosed = CAVE_DETECTOR.debounce(found);
        AbyssMapState.caveRoofFound = found;
        if (found) AbyssMapState.caveRoofWorldY = roofWorldY;
    }
```
